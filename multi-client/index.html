Chess Web UI - Multi-Client Demo

Explore two distinct client implementations for the same chess server backend. This demonstration showcases the flexibility and versatility of the chess engine API.

About the Chess Server: The backend chess server provides a REST API for managing chess games with support for both human and computer players. It features a custom chess engine with configurable difficulty levels and search depths, move validation, game state management, and PGN export capabilities.

The server exposes endpoints for creating games, making moves, undoing moves, and retrieving game state. Both clients below connect to the same server instance, demonstrating how different user interfaces can interact with a unified backend service.

Web Client

Interactive JavaScript-based Chess Interface - A modern web application built with vanilla JavaScript, HTML5, and CSS3. This client provides a rich graphical interface with real-time game state updates and visual feedback.

  • Drag-and-drop piece movement with visual highlights
  • Real-time server health monitoring and status indicators
  • Move history display with PGN notation export
  • Configurable game settings (player color, difficulty, search time)
  • Custom FEN position support for non-standard starting positions
  • Responsive design with mobile support
  • Animated move feedback and checkmate indicators

WASM Terminal Client

Command-line Interface Compiled to WebAssembly - A Go-based terminal client that provides a traditional CLI experience directly in the browser. The Go chess client is compiled to WASM and runs in an xterm.js terminal emulator.

  • Full terminal experience with xterm.js emulation
  • Command-based interface (type "help" to see available commands)
  • ASCII board rendering with ANSI color support
  • Session management with persistent game state
  • Support for authentication, game creation, and move execution
  • Debug commands for API exploration
  • Native Go performance through WebAssembly compilation

Try commands: help, new white 10 1000, move e2e4, board
